HTML Markup
HTML is the heart of the web. HTML makes up everything you can see on a webpage and links to other webpages (which makes the web webby).
Tags like "em" and "b" add some style to the fonts that appear on the webpage.
Attributes are used to add links to the page using "a href" and closing with "a"
Perhaps the easiest way to relieve frustration when typing text in HTML is the Break and Pagraph tags
If it weren't for these two tags, all the text on every webpage would run together like crazy and be super hard to read.
The internet would certainly not be as nice to see if all the text was smushed together.
One other important note about HTML is that there is a standard for opening and closing HTML files so the internet can read them. This is a standard for all webpages on the internet.
